# CUPS Customization

Customização visual da interface web nativa do CUPS para o projeto Print Server ADCETEI.

## Objetivo

Este projeto guarda uma camada visual customizada para o CUPS, sem alterar a lógica de impressão.

A customização é focada em:

- cabeçalho;
- rodapé;
- identidade visual;
- cores;
- espaçamento;
- aparência geral da interface web.

## Importante

Este projeto não altera:

- lógica do CUPS;
- permissões;
- configuração de rede;
- `cupsd.conf`;
- formulários administrativos;
- funcionamento do Email2Print.
